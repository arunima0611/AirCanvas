# Air Canvas: Contactless Gesture-Based Drawing Tool

An AI-powered computer vision application that allows users to draw on a digital canvas in real-time using hand gestures. By leveraging **OpenCV** for frame processing and **MediaPipe** for high-fidelity hand landmark tracking, this project transforms any standard webcam into an interactive, markerless canvas.

---

## 🚀 Features

- **Contactless Drawing:** Draw seamlessly in the air using your index finger.
- **Gesture Control System:** - Raise your **index finger** to draw.
  - Raise both your **index and middle fingers** to toggle "Selection Mode" to change colors or clear the canvas.
- **Dynamic Color Palette:** Switch between vibrant colors (Blue, Green, Red, Yellow) instantly via on-screen UI buttons.
- **Canvas Reset:** Clear the entire screen with a dedicated on-screen clear action.
- **Real-Time Processing:** Low-latency performance optimized for standard webcams.

---

## 🛠️ Tech Stack & Libraries

- **Language:** Python 3.x
- **Computer Vision:** OpenCV (`opencv-python`)
- **Machine Learning / Tracking:** Google MediaPipe (`mediapipe`)
- **Numerical Processing:** NumPy (`numpy`)
